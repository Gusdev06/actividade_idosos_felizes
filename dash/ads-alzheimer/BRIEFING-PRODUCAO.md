# Briefing de produção dos 3 anúncios

> Shot-by-shot com **visual + música + legenda + VO** sincronizados. Para gravar com smartphone, sem produção pesada. Atriz e locação únicas servem para os 3 hooks.

---

## 📐 Especificações técnicas (para os 3 anúncios)

| Item | Valor |
|---|---|
| **Aspect ratio** | 9:16 (1080×1920px) — Reels/Stories/TikTok |
| **Frame rate** | 30 fps (24 fps se quiser look mais "cinema") |
| **Câmera** | Smartphone com câmera traseira + estabilizador (gimbal ou tripé). NUNCA câmera frontal. |
| **Resolução** | 1080p mínimo, 4K se for cortar/zoom no edit |
| **Áudio do set** | Gravar VO em ambiente silencioso (closet/sala fechada), microfone de lapela ou shotgun. NÃO use o áudio do set para a voz — só para som ambiente residual. |
| **Iluminação** | Luz natural lateral (janela à esquerda ou direita da atriz). Cortina branca para difusão. Evite luz frontal achatada. |
| **Hora ideal** | 9h-11h ou 16h-18h (golden hour) |
| **Atriz** | Mulher 48-58 anos, expressão genuína, sem maquiagem pesada, cabelo natural com fios brancos visíveis. Roupa casual (suéter, blusa simples), tons neutros (bege, cinza, azul escuro). |
| **Locação** | Casa real — cozinha, sala ou quarto. NÃO use estúdio. Detalhes do cotidiano no fundo (xícara, livros, foto de família). |
| **Paleta visual** | Tons quentes (bege, ocre, marrom claro). Evite saturação fria/azulada. |
| **Tipografia legendas** | Inter, Montserrat ou Helvetica Bold. Branco com sombra preta sutil (opacidade 50%). Tamanho 80-100px. Posicionamento no terço superior (não cobrir o rosto). |
| **Cor de destaque das legendas** | Roxo `#7c3aed` ou amarelo `#fbbf24` para palavras-chave ("dejó de reconocerme", "no es tu culpa") |

### Música

- **Estilo:** Piano solo melódico, andamento lento (60-70 BPM), tom menor (Am, Em).
- **Track sugerida:** busca em Pixabay Music por "*sad slow piano emotional*" ou "*reflective piano*".
- **Volume:** -24 dB durante a VO, sobe para -14 dB no end card.
- **Sincronização:** entra no segundo 0.5, faz crescendo sutil entre 5-8s, mantém até o final.
- **NÃO usar:** trilhas épicas com strings + drums, EDM, música cristã/gospel óbvia.

### Captions — formatação detalhada

```
Fonte: Inter Bold (ou Montserrat Black)
Tamanho: 92px (mobile-safe)
Cor primária: #FFFFFF (branco)
Cor destaque: #7c3aed (roxo) ou #fbbf24 (amarelo)
Stroke: 4px #000000 (preto sólido)
Shadow: 0 4px 8px rgba(0,0,0,0.5)
Posição: 22% do topo (não cobrir rosto)
Animação: fade in 200ms, fade out 200ms
Sincronização: palavra a palavra (karaoke style) ou frase inteira
```

- **Estilo:** Piano ambient + leve reverb. Tom menor que vira maior no momento "No es tu culpa".
- **Track sugerida:** "*ambient piano warm forgiveness*" — clima de alívio, não tristeza.
- **Volume:** -25 dB durante VO. **PAUSA TOTAL** da música 0.5s antes de "No es tu culpa" (impacto). Música volta suave após.
- **NÃO usar:** percussão, drums, qualquer som rítmico forte.

### Captions — formatação

```

- Texto branco com leve glow #ffffff80
- Sem stroke duro — usar shadow 0 0 20px rgba(0,0,0,0.6) (suave)
- Fonte: Inter Medium (não Bold — tom menos agressivo)

- Letter-spacing: 0.02em (respiro entre letras)
- Posição: 30% do topo
```

## 📋 Checklist antes de gravar

- [ ] Atriz testou o roteiro 3x e está com a fala natural (não decorada palavra-por-palavra)
- [ ] Áudio gravado em closet/sala fechada COM lapela (não com mic do celular)
- [ ] Bateria do celular >70% (estabilização AI consome bateria)
- [ ] Modo cinematic OFF (não queremos blur artificial), HDR ON
- [ ] Limpa a lente do celular (importante!)
- [ ] Locação tem objetos pessoais visíveis no fundo (xícara, foto antiga, planta) — autenticidade
- [ ] Roupa da atriz: sem listras finas (faz moiré), sem branco puro (estoura), sem logos
- [ ] Mãe/pai na cena: se for atriz, não maquiar de forma "doente". Naturalmente envelhecida está perfeito.
- [ ] Caderno do produto impresso fisicamente — comprar uma encadernação na papelaria (~US$3). Vai aparecer várias vezes nos vídeos.

## 🛠 Software sugerido para o edit

- **Mobile:** CapCut Pro (legendas automáticas em espanhol + AI captions + curves)
- **Desktop:** DaVinci Resolve (grátis) — color grading sério + Fairlight pra mix de áudio
- **Legendas auto:** CapCut → Auto Captions → revisar manualmente → estilizar
- **Color grade:** levantar shadows +10, lower highlights -5, saturação -8 (look "documentário")

## 💡 Variações para escalar quando 1 hook ganhar

Quando um hook bater >2.5x ROAS por 3 dias seguidos, criar **5 variações** da mesma estrutura:

- **V1:** mesma atriz, troca cenário (cozinha → quarto da mãe)
- **V2:** mesma atriz, troca primeira frase do VO ("El día que mi padre olvidó mi cumpleaños...")
- **V3:** atriz diferente (45 anos vs 58 anos)
- **V4:** mesma cena, mas perspectiva do filho homem
- **V5:** mesma cena, com depoimento real ao invés de roteiro (UGC influencer parcerizada)

Mantenha a **música igual** entre variações do mesmo hook — facilita reconhecimento de marca.
